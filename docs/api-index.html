TexasHoldem API Reference

Complete API documentation for the TexasHoldem poker game project. This is a single-player Texas Hold'em implementation with AI opponents.

v2.core - Core Game Logic

Core data structures and game logic for Texas Hold'em poker

Suit

Card suit enumeration

class Suit(Enum)

Values: HEARTS, DIAMONDS, CLUBS, SPADES

Rank

Card rank enumeration

class Rank(Enum)

Values: TWO through ACE (2-14)

Card

Represents a playing card with suit and rank

class Card(suit: Suit, rank: Rank)

Methods: __str__, __repr__, __eq__, __hash__

Hand

Represents a poker hand (2 cards)

class Hand(cards: List[Card])

Methods: add_card(), is_full(), __str__

HandRank

Poker hand ranking enumeration

class HandRank(Enum)

Values: HIGH_CARD, ONE_PAIR, TWO_PAIR, THREE_OF_A_KIND, STRAIGHT, FLUSH, FULL_HOUSE, FOUR_OF_A_KIND, STRAIGHT_FLUSH, ROYAL_FLUSH

HandEvaluation

Result of hand evaluation

class HandEvaluation(rank: HandRank, value: int, description: str)

ActionType

Player action types

class ActionType(Enum)

Values: FOLD, CHECK, CALL, BET, RAISE, ALL_IN

Action

Represents a player action

class Action(action_type: ActionType, amount: int = 0)

Phase

Game phase enumeration

class Phase(Enum)

Values: PRE_FLOP, FLOP, TURN, RIVER, SHOWDOWN

SeatStatus

Player seat status

class SeatStatus(Enum)

Values: ACTIVE, FOLDED, ALL_IN, OUT, SITTING_OUT

Player

Represents a poker player

class Player(name: str, chips: int, seat_number: int)

Key methods: bet(), fold(), check(), call(), raise_bet(), go_all_in()

Pot

Manages the poker pot and side pots

class Pot()

Methods: add_bet(), create_side_pot(), distribute_winnings()

Deck

Standard 52-card deck

class Deck()

Methods: shuffle(), deal_card(), reset()

Board

Community cards on the table

class Board()

Methods: add_card(), get_cards(), clear()

HandEvaluator

Evaluates poker hands

class HandEvaluator()

Methods: evaluate_hand(), compare_hands(), get_best_hand()

v2.controller - Game Controller

Main game controller managing game flow and state

GameController

Main game controller class

class GameController(players: List[Player], small_blind: int, big_blind: int)

Key methods: start_game(), play_hand(), handle_betting_round(), determine_winner()

BettingRound

Manages a single betting round

class BettingRound(players: List[Player], current_bet: int)

Usage Examples

Basic Game Setup

# Create players
players = [Player("Human", 1000, 0), Player("AI1", 1000, 1)]

# Start game
controller = GameController(players, small_blind=10, big_blind=20)
controller.start_game()

Hand Evaluation

# Evaluate a hand
evaluator = HandEvaluator()
hand = Hand([Card(Suit.HEARTS, Rank.ACE), Card(Suit.SPADES, Rank.ACE)])
board = Board([Card(Suit.HEARTS, Rank.KING), ...])
result = evaluator.evaluate_hand(hand, board)
